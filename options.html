Job Saver Settings

Configure your API keys and Notion database below.

API Keys

Get yours at console.anthropic.com → API Keys.

Go to notion.so/my-integrations → New integration → copy the token. Then open your Notion database, click ⋯ → Connections, and add your integration.

Notion Database

Open your database in Notion. The URL looks like:
notion.so/your-workspace/DATABASE_ID?v=...
Copy the 32-character ID before the ?. Dashes are optional — both formats work.

Column Names

These must match the exact property names in your Notion database (case-sensitive).
